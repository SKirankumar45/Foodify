🍽️ Foodify — Online Food Ordering Web App

A full-stack food ordering web application built using Django, HTML, and CSS, featuring separate login systems for Admin and Users.
Admins can efficiently manage restaurants and menus, while users can browse restaurants, view menus, add items to their cart, and securely pay using Razorpay.

🚀 Features
👨‍💼 Admin Features

Login and authentication for admin.

Add, update, or remove restaurants.

Manage restaurant menus (add, edit, delete menu items).

View all registered users and their orders.

Dashboard for managing restaurant data and menu items.

🍔 User Features

User registration and login system.

Browse available restaurants and their menus.

Add menu items to the shopping cart.

Place orders and make payments securely using Razorpay integration.

View order history and payment details.

🛠️ Tech Stack
Layer	Technology
Backend	Django (Python)
Frontend	HTML, CSS
Database	SQLite (default)
Payment Gateway	Razorpay
Authentication	Django’s built-in auth system
